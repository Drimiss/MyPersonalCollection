# MyPersonalCollection
Небольшой бот для регулирования потребляемого контента.
## Его возможности
1. Принимать файлы в формате .epub с сайта ficbook
   1. При обработке файлов происходит обновление основной таблицы в столцах B, D, F, J
   2. При добавлении некоторые строки обрабатываются и подаются в таблицу сразу в измененном формате
2. Случайный образом выбирать лист и строку для вывода функция (/random)
3. Производит обновление в выбранном листе по введенному названию (/update)
4. Производит добавление в выбранный лист (/add)
5. Производит сравнение двух указанных в коде листов,чтобы исключить дубликаты из основного листа (/del)
6. Выводит все названия с выбранного листа (/search)
